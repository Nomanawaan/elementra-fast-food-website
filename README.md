📖 Project Overview
Elementra Fast Food Website is a fully responsive restaurant-themed website designed for a fast food business. 
This project was created as part of a web development learning journey and focuses on clean UI/UX, responsive design, and smooth user interactions using core front-end technologies.

🌐 Live Preview
🔗 https://elementra-fast-food-website.vercel.app/

✨ Features
✅ Responsive layout for desktop, tablet and mobile.

✅ Hero section with call-to-action.

✅ Interactive menu section to showcase food items.

✅ About us and services information.

✅ Testimonials carousel or section

✅ Contact or booking form

✅ Fixed navigation menu

✅ Animated scroll effects and transition

✅ Custom footer with social links and copyright
